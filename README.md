# High-Mae (海魅)

High-Mae 是一款专为 Windows 设计的现代化、高性能网络代理客户端。它结合了极简的任务栏托盘操作与功能丰富的 Web 控制面板，基于 sing-box 核心并集成了多种先进协议，旨在提供极致的上网体验与系统资源优化。

## ✨ 核心特性

- **🚀 多协议支持**: 内置对 Hysteria2, AnyTLS, Mieru, SOCKS5-TLS 等高性能协议的原生支持。
- **🎨 双重交互模式**:
    - **托盘菜单**: 快速切换节点、更改路由模式（规则/全局）、开启 TUN 模式及系统代理。
    - **Web 控制面板**: 提供直观的节点管理、订阅导入、实时网速监控及测速功能（默认端口 `10809`）。
- **🔌 强力 TUN 模式**: 集成 `wintun` 与 `tun2socks`，实现真·全局流量接管，支持各种不遵循系统代理设置的应用程序。
- **🛡️ 隐私保护**: 内置 **WebRTC 泄漏防护** 功能，防止真实 IP 通过浏览器泄露。
- **📉 极致内存优化**: 针对 Go 运行时进行了深度调优（GOMEMLIMIT、激进 GC 策略），并将嵌入的二进制文件（tun2socks, wintun）在启动后从内存释放，保持极低的常驻内存占用。
- **⚙️ 智能分流**: 灵活的规则引擎，支持基于域名、IP 及 GeoIP 的智能路由。
- **📋 便捷订阅**: 支持从剪贴板一键解析并导入多种格式的节点订阅。

## 🛠️ 技术架构

- **核心引擎**: 基于 [sing-box](https://github.com/sagernet/sing-box) (及定制版 mbox)。
- **界面逻辑**: 使用 `getlantern/systray` 实现原生 Windows 托盘；使用 **Go 标准库 (net/http)** 构建极简 Web API，不引入额外路由框架。
- **网络底层**: 自研 `protocol` 与 `ins` (Internal) 模块，处理复杂的连接转换与系统配置。
- **构建系统**: 单可执行文件设计，自动释放并管理 `wintun.dll` 等依赖。

## 🚀 快速开始

### 运行环境
- Windows 10/11 (x64/ARM64)
- 管理员权限（开启 TUN 模式时需要）

### 获取与运行
1. 下载最新的 [Releases](https://github.com/Ethan-Wanng/high-mae/releases) 版本。
2. 直接运行 `HighMae.exe`。
3. 程序启动后会自动在默认浏览器打开控制面板 `http://127.0.0.1:10809/`。

### 本地构建
如果你想从源码构建：
```powershell
# 确保已安装 Go 1.25+
go mod download
./mk.ps1  # 使用内置的 PowerShell 脚本进行编译
```

## 📖 使用说明

1. **导入节点**: 在 Web 控制面板点击「导入节点/订阅」，粘贴你的订阅链接或节点分享链接。
2. **切换模式**: 
   - **规则分流**: 仅代理墙外流量，不影响国内速度。
   - **全局代理**: 所有请求均通过代理节点。
3. **TUN 模式**: 当需要加速游戏、UWP 应用或命令行工具时，在托盘菜单开启「虚拟网卡 (TUN)」。

## ⚠️ 开发注意事项

- **内存限制**: 程序默认设置了 `GOMEMLIMIT=150MiB`，以确保在任务管理器中的内存显示始终处于较低水平。
- **二进制依赖**: `tun2socks.exe` 和 `wintun.dll` 是以 `//go:embed` 形式打包在主程序中的，首次运行会自动释放到当前目录。
